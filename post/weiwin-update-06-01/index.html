微信最强更新，微信号可以改名了

2020-06-01 2 min read # 微信 # 更新 # 改名

微信最强更新，微信号可以改名了！方法如下

以往，微信号改名界面上会提示【只能设置一次】。

不过在 7.0.15 中变为了【一年设置一次】。

更改微信号需要满足 3 个条件：

  • 微信号更改超过 1 年

  • 最近 2 周没有改过密码。

  • 在同一设备上登录 1 周以上。

但是各位先不要高兴得太早。

这个功能刚刚出现，目前还处于灰度测试中。

只有一小部分人可以更改，并不是每个人都能更改的。

如果你之前更改过微信号，正常情况下，大部分人微信号那一栏是点不进去的。

不过 IT 之家 的网友提供了一个特殊手段：

如果你的手机 root 过，可以通过以下步骤 “激活” 改名通道。

改名步骤

先在【IT 之家】微信公众号后台回复【微信改名】，获取最新版微信安装包和工具。

都安装好后，打开工具。

在上面的搜索框里搜索【微信】，点击【活动列表】。

然后点击右上角的【┇】，选择【也显示未导出的活动】。

之后再次搜索：

com.tencent.mm.plugin.setting.ui.setting.SettingsAliasUI

找到这个活动后，点击右侧的【详情】。

再点打开。

这时就可以改名了。

当然，还是要再强调一遍：

  • 微信改名目前正处于灰度测试阶段，不是每个人都能改。

  • 改名要满足文章开头的 3 个硬性条件。

  • 如果没被选中，使用特殊方法时，要确保手机拥有 root 权限。
