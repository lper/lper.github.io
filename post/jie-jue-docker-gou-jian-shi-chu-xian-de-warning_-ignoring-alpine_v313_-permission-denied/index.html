解决 Docker 构建时出现的 WARNING: Ignoring https://dl-cdn.alpinelinux.org/alpine/v3.13/main: Permission denied

在根据官网的 Docker-compose 构建示例项目时遇到的问题，原因是因为无法访问官网给出的外网镜像，可能还会出现 WARNING: Ignoring https://dl-cdn.alpinelinux.org/alpine/v3.13/main: network error
通过在 Dockerfile 中引入下面两个命令可以解决该问题：

RUN sed -i ‘s/https/http/’ /etc/apk/repositories
RUN apk add curl
